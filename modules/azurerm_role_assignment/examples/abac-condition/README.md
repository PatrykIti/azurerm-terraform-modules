# ABAC Condition Example

This example assigns a role with an attribute-based access control (ABAC) condition.

## Features

- Uses `condition` and `condition_version = "2.0"`
- Demonstrates role assignment at a storage account scope
- Creates a user-assigned managed identity to receive the assignment

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
