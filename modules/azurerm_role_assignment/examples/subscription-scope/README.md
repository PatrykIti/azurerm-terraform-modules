# Subscription Scope Example

This example assigns the built-in Reader role at the subscription scope.

## Features

- Uses subscription scope via `azurerm_subscription` data source
- Creates a user-assigned managed identity for the assignment

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
