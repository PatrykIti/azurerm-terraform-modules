# Basic Role Assignment Example

This example assigns the built-in Reader role at a resource group scope to a user-assigned managed identity.

## Features

- Creates a resource group and user-assigned managed identity
- Assigns the Reader role at the resource group scope
- Uses a fixed naming convention for repeatable runs

## Key Configuration

This example demonstrates the minimal inputs required for a role assignment module.

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
