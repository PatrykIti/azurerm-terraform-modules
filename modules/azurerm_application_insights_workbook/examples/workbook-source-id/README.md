# Workbook Source ID Example

This example demonstrates an Application Insights Workbook referencing a source
resource via `source_id`.

## Features

- Log Analytics workspace as the source resource
- Workbook configured with `source_id`

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
