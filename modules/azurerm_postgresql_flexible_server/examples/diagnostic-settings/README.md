# Diagnostic Settings PostgreSQL Flexible Server Example

This example demonstrates streaming PostgreSQL Flexible Server logs and metrics
into a Log Analytics workspace using diagnostic settings.

## Features

- Log Analytics workspace for diagnostics.
- Diagnostic settings enabled with all available categories.

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
