# PostgreSQL Flexible Server Firewall Rules Example

This example demonstrates public network access with firewall rules for
allow-listed IP ranges.

## Features

- Public access enabled.
- Firewall rules for office and VPN ranges.

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
