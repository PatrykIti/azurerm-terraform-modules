# PostgreSQL Flexible Server Customer-Managed Key Example

This example demonstrates customer-managed key encryption using a user-assigned
identity and an Azure Key Vault key.

## Features

- User-assigned identity for encryption.
- Key Vault and key provisioning.
- Customer-managed key configuration.

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
