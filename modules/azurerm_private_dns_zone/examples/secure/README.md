# Secure Private DNS Zone Example

This example focuses on a hardened Private DNS Zone deployment with clear ownership tagging.

## Features

- Uses a dedicated resource group
- Applies security and classification tags
- Ready to be linked only to trusted VNets via the Virtual Network Link module

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
