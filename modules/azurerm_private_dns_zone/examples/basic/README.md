# Basic Private DNS Zone Example

This example provisions a minimal Private DNS Zone with tags and a dedicated resource group.

## Features

- Creates a Private DNS Zone with a simple domain name
- Uses a dedicated resource group
- Applies basic tagging

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
