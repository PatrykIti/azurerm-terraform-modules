# Basic App Service Plan Example

This example demonstrates a minimal App Service Plan deployment using a shared
Windows SKU with only the required module inputs.

## Features

- Creates a dedicated resource group
- Deploys a basic Windows App Service Plan
- Uses the local module source for development and validation

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
