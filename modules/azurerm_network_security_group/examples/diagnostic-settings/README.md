# Diagnostic Settings Network Security Group Example

This example demonstrates configuring NSG diagnostic settings with multiple destinations.

## Features

- **NSG diagnostics** for events, rule counters, and metrics
- **Log Analytics** for centralized log queries
- **Storage Account** for long-term log retention
- **Event Hub** for streaming logs to external systems

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
