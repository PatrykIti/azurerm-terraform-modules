# Complete Azure DevOps Group Example

This example demonstrates group creation with membership composition.

## Features

- Creates a primary Azure DevOps group managed by the module
- Creates a secondary group used as membership source
- Adds strict-child membership with stable key

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
