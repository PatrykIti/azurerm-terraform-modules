# Secure Azure DevOps Group Example

This example demonstrates a security-focused group configuration with explicit membership overwrite mode.

## Features

- Creates dedicated security group
- Adds explicit membership with deterministic key
- Uses overwrite mode for strict membership control

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
