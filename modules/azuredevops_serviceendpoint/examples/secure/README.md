# Secure Azure DevOps Service Endpoints Example

This example demonstrates a repository with stricter review and status policies.

## Features

- Minimum reviewers policy
- Status check policy
- Credential scanning and case enforcement policies

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
