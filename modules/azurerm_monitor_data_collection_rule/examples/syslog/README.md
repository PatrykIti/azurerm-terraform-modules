# Syslog Monitor Data Collection Rule Example

This example demonstrates a Data Collection Rule configured for Linux syslog data.

## Features

- Creates a DCR configured for syslog collection
- Sends syslog streams to Log Analytics
- Uses Linux-specific data sources

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
