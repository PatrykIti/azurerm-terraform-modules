# Complete Role Definition Example

This example demonstrates a custom role definition with management and data actions.

## Features

- Uses `actions`, `not_actions`, `data_actions`, and `not_data_actions`
- Adds a description for operational clarity
- Uses subscription scope for assignable scopes

## Key Configuration

This example focuses on core RBAC capabilities that are commonly required for custom roles.

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
