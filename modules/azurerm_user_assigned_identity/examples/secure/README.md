# Secure User Assigned Identity Example

This example demonstrates a User Assigned Identity with a federated identity credential and least-privilege RBAC applied outside the module.

## Features

- User Assigned Identity with tags
- Federated identity credential (OIDC)
- Least-privilege role assignment at the resource-group scope

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
