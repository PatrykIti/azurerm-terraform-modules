# Complete User Assigned Identity Example

This example demonstrates a User Assigned Identity with federated identity credentials and custom timeouts.

## Features

- User Assigned Identity with tags
- Multiple federated identity credentials (OIDC)
- Custom timeouts

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
