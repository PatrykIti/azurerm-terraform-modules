# Federated Identity Credentials Example

This example focuses on configuring federated identity credentials for a User Assigned Identity.

## Features

- User Assigned Identity with multiple federated identity credentials
- Common OIDC issuer patterns (GitHub Actions and Kubernetes)

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
