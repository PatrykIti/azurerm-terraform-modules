# GitHub Actions OIDC Example

This example shows how to configure a federated identity credential for GitHub Actions
OIDC tokens.

## Features

- User Assigned Identity
- GitHub Actions OIDC federated identity credential
- Variables to customize repository and git ref

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
