# Complete Private DNS Zone Virtual Network Link Example

This example enables registration and sets the resolution policy for the VNet link.

## Features

- Creates a Private DNS Zone and Virtual Network
- Enables `registration_enabled`
- Sets `resolution_policy`
- Applies production tags

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
