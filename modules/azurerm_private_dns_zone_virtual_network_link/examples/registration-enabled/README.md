# Registration Enabled Private DNS Zone Link Example

This example enables auto-registration of VM records in the linked virtual network.

## Features

- Creates a Private DNS Zone and Virtual Network
- Enables `registration_enabled` on the link

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
