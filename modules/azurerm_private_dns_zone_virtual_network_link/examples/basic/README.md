# Basic Private DNS Zone Virtual Network Link Example

This example creates a Private DNS Zone and links it to a Virtual Network.

## Features

- Creates a Private DNS Zone
- Creates a Virtual Network
- Links the zone to the VNet

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
