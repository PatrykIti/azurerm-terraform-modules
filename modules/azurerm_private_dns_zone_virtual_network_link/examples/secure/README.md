# Secure Private DNS Zone Virtual Network Link Example

This example links a Private DNS Zone to a dedicated VNet intended for private endpoints.

## Features

- Dedicated resource group
- Explicitly disables auto-registration
- Tags for ownership and auditing

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
