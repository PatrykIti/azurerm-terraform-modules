# Basic Windows Virtual Machine Example

This example demonstrates a basic Windows VM deployment with a public IP and
minimal OS disk configuration.

## Features

- Windows VM with a generated admin password
- Virtual network, subnet, public IP, and NIC
- Minimal OS disk configuration

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
