# Basic Cognitive Account Example (OpenAI)

This example provisions a minimal OpenAI Cognitive Account with public access enabled.

## Features

- OpenAI account with SKU S0
- Public network access enabled
- Local authentication enabled (keys)

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
