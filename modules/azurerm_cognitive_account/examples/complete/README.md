# Complete Cognitive Account Example (OpenAI)

This example demonstrates a more complete OpenAI configuration including network ACLs, identity, and diagnostic settings.

## Features

- OpenAI account with custom subdomain
- Network ACLs with VNet rule
- User-assigned managed identity
- Diagnostic settings to Log Analytics

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
