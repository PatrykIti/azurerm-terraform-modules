# Secure Event Hub Namespace Example

This example demonstrates a security-focused Event Hub Namespace configuration.

## Features

- Public network access disabled
- Customer-managed key (CMK) with user-assigned identity
- Private endpoint for namespace access

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
