# Network Rule Set Event Hub Namespace Example

This example demonstrates configuring an Event Hub Namespace with a network rule set (IP rules and VNet rules).

## Features

- Event Hub Namespace with Standard SKU
- Network rule set with IP allow list and VNet subnet rule
- Service endpoint-enabled subnet

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
