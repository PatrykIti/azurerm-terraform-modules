# Secure Azure Monitor Private Link Scope Example

This example demonstrates a secure Azure Monitor Private Link Scope (AMPLS) configuration with PrivateOnly access modes.

## Features

- PrivateOnly ingestion and query access
- Scoped services limited to a Log Analytics workspace

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
