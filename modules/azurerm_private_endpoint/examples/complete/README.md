# Complete Private Endpoint Example

This example demonstrates a fuller configuration, including a custom NIC name, static IP configuration, and a private DNS zone group.

## Highlights

- Custom network interface name
- Static IP configuration (`ip_configurations`)
- Private DNS zone and VNet link
- DNS zone group attachment on the Private Endpoint

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
