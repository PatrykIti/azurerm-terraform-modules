# Basic Bastion Host Example

This example demonstrates a basic deployment of the Bastion Host module using secure defaults.

## Features

- Creates a Basic SKU Bastion Host.
- Provisions the required `AzureBastionSubnet` and a Standard public IP.

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
