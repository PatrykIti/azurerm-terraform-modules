# Complete Bastion Host Example

This example demonstrates a Standard SKU Bastion Host with advanced features and diagnostic settings.

## Features

- Standard SKU with scale units.
- Feature toggles enabled: file copy, IP connect, shareable link, tunneling, and Kerberos.
- Diagnostic settings routed to Log Analytics.

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
