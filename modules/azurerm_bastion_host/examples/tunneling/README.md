# Tunneling Bastion Host Example

This example demonstrates enabling native client tunneling on a Standard SKU Bastion Host.

## Features

- Standard SKU Bastion Host with tunneling enabled.
- Required Bastion subnet and public IP.

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
