# Diagnostic Settings Bastion Host Example

This example demonstrates configuring Azure Monitor diagnostic settings for a Bastion Host.

## Features

- Standard SKU Bastion Host with diagnostic settings.
- Sends audit logs and metrics to Log Analytics using `areas` mapping.

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
