# Shareable Link Bastion Host Example

This example demonstrates enabling Shareable Link on a Standard SKU Bastion Host.

## Features

- Standard SKU Bastion Host with shareable links enabled.
- Required Bastion subnet and public IP.

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
