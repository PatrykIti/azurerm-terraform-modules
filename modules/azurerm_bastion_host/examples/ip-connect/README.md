# IP Connect Bastion Host Example

This example demonstrates enabling IP Connect on a Standard SKU Bastion Host.

## Features

- Standard SKU Bastion Host with IP Connect enabled.
- Required Bastion subnet and public IP.

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
