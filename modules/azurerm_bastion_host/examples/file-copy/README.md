# File Copy Bastion Host Example

This example demonstrates enabling file copy on a Standard SKU Bastion Host.

## Features

- Standard SKU Bastion Host with file copy enabled.
- Required Bastion subnet and public IP.

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
