# Complete Linux Function App Example

This example demonstrates a richer Linux Function App configuration including
application settings, auth settings (v1), diagnostics, and App Insights.

## Features

- App settings and connection strings
- Auth settings (v1) with Azure AD
- Site configuration with TLS, CORS, health checks, and IP restrictions
- Diagnostic settings to Log Analytics
- Application Insights integration

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
