# Terraform Azure Linux Function App Module

## Module Version

<!-- BEGIN_VERSION -->
Current version: **1.0.0**
<!-- END_VERSION -->

## Description

Manages Azure Linux Function Apps with optional slots and diagnostic settings.

## Usage

```hcl
module "linux_function_app" {
  source = "path/to/azurerm_linux_function_app"

  name                = "example-func"
  resource_group_name = azurerm_resource_group.example.name
  location            = azurerm_resource_group.example.location

  service_plan_id = azurerm_service_plan.example.id

  storage_configuration = {
    account_name       = azurerm_storage_account.example.name
    account_access_key = azurerm_storage_account.example.primary_access_key
  }

  site_configuration = {
    application_stack = {
      node_version = "20"
    }
  }

  tags = {
    Environment = "Development"
    Project     = "Example"
  }
}
```
