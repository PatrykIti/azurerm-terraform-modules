# Complete Azure DevOps Group Entitlement Example

This example creates one entitlement using the `origin+origin_id` selector mode.

## Features

- Full selector configuration (`origin` and `origin_id`)
- Configurable license fields

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
