# Secure Azure DevOps Group Entitlement Example

This example applies a minimal-privilege entitlement profile using `stakeholder` license.

## Features

- Explicit entitlement key
- Conservative license profile for broad-read scenarios

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Cleanup

```bash
terraform destroy
```
